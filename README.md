<h1 align="center">Storage & Ethernet System</h1>
<p align="center">
A compact PCB designed to provide redundant onboard storage and wired Ethernet connectivity for embedded systems where robustness, data integrity and easy integration are key.

### Overview

The main goal of this PCB is to improve the reliability of embedded data logging and communication systems.

It is especially useful in applications where removable media such as microSD cards may be affected by vibration, mechanical stress, poor contact or harsh environmental conditions.

The board is designed to support:

- Reliable onboard data storage.
- Redundant logging.
- Wired Ethernet communication.
- Compact integration.
- Post-operation data recovery.

### Main Features

- Dual SPI NOR Flash memories.
- Independent chip-select lines for each memory.
- WIZnet W5500 hardware TCP/IP Ethernet controller.
- SiTime SiT5000 25 MHz TCXO clock source.
- SPI interface with the main controller.
- ESD-aware external interface design.
- Ethernet analog supply isolation.
- Custom KiCad symbols, footprints and 3D models.
